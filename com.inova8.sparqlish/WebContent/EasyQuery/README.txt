EasyQuery.JS
============


What is it?
--------------
EasyQuery.JS is a set of jQuery widgets which allows you to implement an ad-hoc, user-friendly SQL query bulider in your web application.
You can find more information about EasyQuery.JS here: http://devtools.korzh.com/easyquery/javascript/



How to start
----------------
It's simple:

1. Oppen "Getting started" web-page: http://devtools.korzh.com/easyquery/javascript/docs/javascript-query-builder-php 

2. Follow the instructions listed there.
